2.skill.json
id:id
name:名字
explain:说明
minLevel:最小激活等级
m_type:伤害类型
target:作用目标
first:优先作用对象
number:作用数量
range_type:作用类型
range: x|y|w|h 作用范围  如果是范围技能可以使用,之前都是根据数量匹配角色
multiple:效果倍率   0.生命|1.物攻|2.物抗|3.法攻|4.法抗|5.暴击|6.抗暴|7.攻击速度|8.移动速度|9.攻击范围

rounds:作用回合
